# Hack4Delhi - Railway AI Safety System 🚆

![Railway AI Banner](railway_ai_system(version%202)/railway_ai_system/images/banner.png)

Welcome to the **Railway AI Safety System**, built for the Hack4Delhi Hackathon. This repository contains the complete implementation of a next-generation AI-powered railway monitoring and alert system.

We have developed the system in two major iterations:

## 🔹 [Version 1: Streamlit Prototype](./railway_ai_system(version%201)/)
The initial prototype built quickly using Python and Streamlit to prove the core concept of AI-driven defect detection on railway tracks.
- Uses Streamlit for the Control Room Dashboard.
- Incorporates basic Vision, Sound, and Vibration detection logic.

## 🔹 [Version 2: Full-Stack React & FastAPI Application](./railway_ai_system(version%202)/)
The final production-ready system featuring a scalable architecture.
- **Backend:** FastAPI (Python) for high-performance REST APIs.
- **Frontend:** React + Vite for a blazing fast, responsive user interface.
- **AI Models:** YOLOv8 integration for real-time track anomaly and human intrusion detection.
